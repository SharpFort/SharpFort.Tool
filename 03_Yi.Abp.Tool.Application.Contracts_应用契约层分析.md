# Yi.Abp.Tool.Application.Contracts - 应用契约层分析

## 📌 概述

**Yi.Abp.Tool.Application.Contracts** 是 ABP 分层架构中的**应用服务契约层**（Application Contracts Layer），定义了应用服务的接口和数据传输对象（DTO）。遵循 DDD 的**依赖倒置原则**——高层模块（CLI 命令）依赖本层的抽象接口，而非具体实现。

**项目路径:** `tool/Yi.Abp.Tool.Application.Contracts/`

## 🏗️ 项目信息

| 属性 | 值 |
|------|-----|
| **TargetFramework** | net8.0 |
| **输出类型** | Class Library（类库） |

## 📦 项目依赖

| 依赖 | 说明 |
|------|------|
| `Yi.Abp.Tool.Domain.Shared` | 共享领域层（枚举、DTO、配置选项） |
| `Yi.Framework.Ddd.Application.Contracts` | Yi框架 DDD 应用契约基础 |

### 2. ITemplateGenService.cs - 模板生成服务接口 ⭐

**文件路径:** `Yi.Abp.Tool.Application.Contracts/ITemplateGenService.cs`

```csharp
public interface ITemplateGenService : IApplicationService
{
    Task<byte[]> CreateModuleAsync(TemplateGenCreateInputDto moduleCreateInputDto);
    Task<List<string>> GetAllTemplatesAsync();
}
```

**接口定义两个方法:**

| 方法 | 返回类型 | 说明 |
|------|----------|------|
| `CreateModuleAsync` | `Task<byte[]>` | 创建模块模板，返回 zip 文件的字节数组 |
| `GetAllTemplatesAsync` | `Task<List<string>>` | 获取所有可用模板名称列表 |

**设计说明:**
- 继承 `IApplicationService`（ABP 框架标记接口），自动注册为应用服务
- 此接口会被 HttpApi.Client 层用于生成**动态 HTTP 代理客户端**

### 3. TemplateGenCreateInputDto.cs - 创建模板输入 DTO

**文件路径:** `Yi.Abp.Tool.Application.Contracts/Dtos/TemplateGenCreateInputDto.cs`

```csharp
public class TemplateGenCreateInputDto
{
    public string Name { get; set; }                                    // 模块名称
    public string ModuleSoure { get; set; }                            // 模板分支来源
    public DbmsEnum Dbms { get; set; }                                 // 数据库提供者
    public Dictionary<string, string> ReplaceStrData { get; set; } = new();  // 字符串替换字典

    public void SetNameReplace()
    {
        ReplaceStrData.Add("Yi.Abp", Name);
        ReplaceStrData.Add("YiAbp", Name.Replace(".", ""));
    }
}
```

**属性详解:**

| 属性 | 类型 | 说明 |
|------|------|------|
| `Name` | `string` | 新模块的名称 |
| `ModuleSoure` | `string` | 模块模板分支来源（对应 Gitee 分支名） |
| `Dbms` | `DbmsEnum` | 数据库类型枚举 |
| `ReplaceStrData` | `Dictionary<string,string>` | 模板内容替换字典 |

**SetNameReplace() 方法:**
```
输入: Name = "MyCompany.MyModule"
结果:
  ReplaceStrData["Yi.Abp"] = "MyCompany.MyModule"
  ReplaceStrData["YiAbp"] = "MyCompanyMyModule"
```
> 此方法在创建模板时，将模板内置的命名空间占位符替换为用户指定的名称。

## 📝 契约层职责边界

| 职责 | 本层 | 非本层 |
|------|:--:|:-----:|
| 定义服务接口 | ✅ | |
| 定义 DTO 数据结构 | ✅ | |
| 引用 Domain.Shared 类型 | ✅ | |
| 实现业务逻辑 | ❌ | Application 层 |
| 持久化操作 | ❌ | Domain 层 |
| HTTP 通信 | ❌ | HttpApi.Client 层 |
