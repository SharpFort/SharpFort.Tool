# Yi.Abp.Tool.Web - Web API 服务端分析

## 📌 概述

**Yi.Abp.Tool.Web** 是 CLI 工具配套的 **Web API 服务端**，基于 ASP.NET Core 8.0，为 CLI 客户端提供远程模板生成等服务。采用 ABP 框架的**动态 API 控制器**自动暴露应用服务为 RESTful API。

**项目路径:** `tool/Yi.Abp.Tool.Web/`

## 🏗️ 项目信息

| 属性 | 值 |
|------|-----|
| **TargetFramework** | net8.0 |
| **项目类型** | ASP.NET Core Web API |
| **默认端口** | `19002` (HTTP) |
| **Swagger 路径** | `/swagger` |
| **API 根路径** | `/api/app` |

## 📦 项目依赖

| 依赖 | 说明 |
|------|------|
| `Yi.Abp.Tool.Application` | 应用服务层 |
| `Yi.Framework.AspNetCore` | Yi 框架 ASP.NET Core 集成 |
| `Volo.Abp.AspNetCore.Mvc` | ABP ASP.NET Core MVC |
| `Volo.Abp.Autofac` | ABP Autofac DI |
| `Volo.Abp.Swashbuckle` | ABP Swagger 集成 |
| `Microsoft.AspNetCore.Mvc.NewtonsoftJson` 8.0.6 | Newtonsoft JSON 支持 |

## 📁 文件结构与功能

### 1. Program.cs - Web 入口

```csharp
var builder = WebApplication.CreateBuilder(args);
builder.WebHost.UseUrls(builder.Configuration["App:SelfUrl"]);
builder.Host.UseAutofac();
await builder.Services.AddApplicationAsync<YiAbpToolWebModule>();
var app = builder.Build();
await app.InitializeApplicationAsync();
await app.RunAsync();
```

**关键配置:**
- 应用 URL 从 `appsettings.json` → `App:SelfUrl` → `http://*:19002`
- 使用 Autofac 作为 DI 容器

#### 📋 服务配置 (ConfigureServicesAsync)

##### A. 动态 API 控制器

```csharp
Configure<AbpAspNetCoreMvcOptions>(options =>
{
    options.ConventionalControllers.Create(
        typeof(YiAbpToolApplicationModule).Assembly, 
        options => options.RemoteServiceName = "tool"
    );
    // 统一前缀
    options.ConventionalControllers.ConventionalControllerSettings
        .ForEach(x => x.RootPath = "api/app");
});
```

**效果:**
- 自动扫描 `YiAbpToolApplicationModule` 程序集中的应用服务
- 为每个服务生成 API 控制器
- 所有 API 统一前缀: `/api/app`

##### B. JSON 序列化配置

##### E. 跨域配置 (CORS)

```csharp
context.Services.AddCors(options =>
{
    options.AddPolicy(DefaultCorsPolicyName, builder =>
    {
        builder
            .WithOrigins(configuration["App:CorsOrigins"]!
                .Split(";", StringSplitOptions.RemoveEmptyEntries)
                .Select(o => o.RemovePostFix("/"))
                .ToArray())
            .WithAbpExposedHeaders()
            .SetIsOriginAllowedToAllowWildcardSubdomains()
            .AllowAnyHeader()
            .AllowAnyMethod()
            .AllowCredentials();
    });
});
```

**允许的来源（从配置）:**
```
App:CorsOrigins = "http://localhost:18001;https://ccnetcore.com"
```

##### F. 速率限制 (Rate Limiting)

```csharp
service.AddRateLimiter(_ =>
{
    _.RejectionStatusCode = StatusCodes.Status429TooManyRequests;
    _.GlobalLimiter = PartitionedRateLimiter.CreateChained(
        PartitionedRateLimiter.Create<HttpContext, string>(httpContext =>
        {
            var userAgent = httpContext.Request.Headers.UserAgent.ToString();
            return RateLimitPartition.GetSlidingWindowLimiter(userAgent, _ =>
                new SlidingWindowRateLimiterOptions
                {
                    PermitLimit = 1000,           // 每窗口最多 1000 个请求
                    Window = TimeSpan.FromSeconds(60),  // 60 秒窗口
                    SegmentsPerWindow = 6,        // 分 6 段滑动
                    QueueProcessingOrder = QueueProcessingOrder.OldestFirst
                });
        }));
});
```

**速率限制策略:**
- 算法: **滑动窗口**（Sliding Window）
- 维度: 按 User-Agent 限制
- 限制: 每 60 秒最多 1000 个请求
- 窗口分段: 6 段（每 10 秒一段）
- 超出限制: 返回 429 Too Many Requests

#### 📋 应用初始化 (OnApplicationInitializationAsync)

```csharp
app.UseRouting();                    // 路由
app.UseCors(DefaultCorsPolicyName);  // 跨域
app.UseRateLimiter();                // 速率限制（非开发环境）
app.UseYiSwagger();                  // Swagger UI
app.UseApiInfoHandling();            // API 信息处理
app.UseStaticFiles("/api/app/wwwroot");     // 静态文件
app.UseDefaultFiles();               // 默认文件
app.UseDirectoryBrowser("/api/app/wwwroot"); // 目录浏览
app.UseConfiguredEndpoints();        // 终节点（动态 API）
```

**中间件管道:**
```
HTTP Request
  → Routing
    → CORS
      → RateLimiter (仅非开发环境)
        → Swagger (/swagger)
        → ApiInfoHandling
        → StaticFiles (/api/app/wwwroot)
        → Dynamic API Endpoints
          → Application Services
```

### 3. appsettings.json - 主配置文件

```json
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*",
  "App": {
    "SelfUrl": "http://*:19002",
    "CorsOrigins": "http://localhost:18001;https://ccnetcore.com"
  },
  "ToolOptions": {
    "TempDirPath": "wwwroot/temp",
    "ModuleTemplateFilePath": "wwwroot/ModuleTemplate.zip",
    "ProjectTemplateFilePath": "wwwroot/ProjectTemplate.zip"
  },
  "GiteeAccession": ""
}
```

**配置项完整说明:**

| 配置路径 | 值 | 说明 |
|----------|-----|------|
| `App:SelfUrl` | `http://*:19002` | 服务监听地址（所有 IP，端口 19002） |
| `App:CorsOrigins` | `http://localhost:18001;https://ccnetcore.com` | 允许的跨域来源 |
| `ToolOptions:TempDirPath` | `wwwroot/temp` | 模板临时文件目录 |
| `ToolOptions:ModuleTemplateFilePath` | `wwwroot/ModuleTemplate.zip` | 模块模板文件（预留） |
| `ToolOptions:ProjectTemplateFilePath` | `wwwroot/ProjectTemplate.zip` | 项目模板文件（预留） |
| `GiteeAccession` | `""` | Gitee API 访问令牌 |
